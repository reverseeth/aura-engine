Guia de instalação completo

Aura AI
BASE DE DADOS
1 Instale o Claude

Assine o plano Pro ou Max em claude.ai

2 Conecte a base de conhecimento

Opção A — Claude Desktop (app)

1. Abra o Claude Desktop
2. Vá em Settings (ícone de engrenagem)
3. Clique em "Integrations"
4. Clique em "+ Add Custom Integration"
5. Nome: Aura
6. URL: cole o endereço abaixo
7. Clique em "Add" — feche e reabra o Claude Desktop

https://aura-mcp-production.up.railway.app/mcp

Opção B — Claude Code (terminal)

Abra o Terminal e cole esses 2 comandos (um de cada vez):

claude mcp add --transport http aura https://aura-mcp-production.up.railway.app/mcp
mkdir -p ~/.claude && echo 'Quando a mensagem começar com "aura:", use a ferramenta aura (search_knowledge) para responder com base nos resultados. Quando a mensagem NÃO começar com "aura:", responda normalmente sem usar a ferramenta aura.' > ~/.claude/CLAUDE.md

Feche e abra o Claude Code. Pronto.

3 Use

Comece sua mensagem com aura: e faça sua pergunta.

aura: qual a melhor estrutura de campanha para Meta Ads?
aura: analisa essa headline: "Perca 10kg em 30 dias"
aura: como montar uma oferta irresistível para um produto de $200?

Mensagens com aura: consultam a base de conhecimento.
Mensagens sem aura: funcionam como Claude normal.

Aura Engine
SKILLS
1 Instale o Claude Code

O Claude Code é a versão terminal do Claude. Rode no Terminal (Mac) ou PowerShell (Windows):

Mac
curl -fsSL https://claude.ai/install.sh | sh
Windows
irm https://claude.ai/install.ps1 | iex
2 Baixe o Aura Engine

Ainda no Terminal, rode:

git clone https://github.com/reverseeth/aura-engine.git cd aura-engine claude
3 Configure

Quando o Claude Code abrir, digite:

setup

Siga as instruções na tela. O sistema configura tudo automaticamente.

O Aura Engine se atualiza sozinho toda vez que você abrir.
Desinstalação
– Claude Desktop

Settings → Integrations → remova Aura

– Claude Code

Cole no terminal (um de cada vez):

claude mcp remove aura
rm ~/.claude/CLAUDE.md
